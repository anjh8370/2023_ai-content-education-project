# AI 기반 음성·영상 콘텐츠 제작 수업 설계 프로젝트

## 📌 프로젝트 개요
AI 도구들을 활용하여 음성 및 영상 콘텐츠를 제작하는 과정을 직접 설계하고, 이를 기반으로 교육 수업을 구성하여 진행한 프로젝트입니다.  
단순한 도구 사용이 아닌, 여러 AI 도구를 연결하여 하나의 결과물을 만들어내는 흐름을 설계하는 데에 초점을 두었습니다.

---

## 🎯 프로젝트 목적
- AI 기반 콘텐츠 제작 과정을 직접 설계
- 교육 수업으로 활용 가능한 형태로 구성
- 다양한 AI 도구를 연결하여 하나의 제작 흐름 구축

---

## 🛠 사용 기술
- ChatGPT (스크립트 생성)
- Beatoven AI (음악 생성)
- Deepbrain AI (영상 생성)
- RVC (음성 변환)
- Gradio / Google Colab
- 온라인 영상 편집 도구

---

## ⚙️ 전체 과정 (Pipeline)

AI 콘텐츠 제작 수업을 위해 아래와 같은 과정을 설계하였습니다.

1. ChatGPT를 활용한 스크립트 생성 및 수정  
2. Beatoven AI를 활용한 배경 음악 생성  
3. Deepbrain AI를 활용한 영상 제작  
4. RVC 기반 음성 변환을 통한 목소리 변경  
5. 영상 및 음성 편집 후 결과물 생성  

---

## 📊 결과
- AI 기반 음성·영상 콘텐츠 제작 수업 진행
- 학생들이 직접 AI 도구를 활용하여 콘텐츠를 제작할 수 있도록 교육
- 단순 사용이 아닌, AI 활용 흐름을 이해할 수 있는 경험 제공

---

## 💡 배운 점
- 여러 AI 도구를 개별적으로 사용하는 것이 아니라, 하나의 흐름으로 연결하는 것이 중요하다는 것을 느꼈습니다.
- 수업을 직접 설계하고 진행하면서, 기술 자체보다 이를 어떻게 구조화하고 전달할 것인지가 더 중요하다는 것을 경험했습니다.
- 음성 변환과 같은 기술이 잘못 사용될 경우 발생할 수 있는 윤리적 문제에 대해서도 고민하게 되었습니다.
